1. Optimized Tech Stack

AWS Cloud, Monolithic & Cost-Effective

To maintain the low budget while shifting to AWS, we must utilize the AWS 12-Month Free Tier. We will deploy a Laravel (PHP) application on EC2, connecting to a managed PostgreSQL/MySQL RDS instance, and store heavy images in S3.

Frontend (Mobile + Web)

Flutter (Mobile) + Laravel Blade (Web)

  • Flutter handles the Android App for Employees (Native GPS/Camera).
  • Laravel Blade (Server-Side) renders Admin, Client, and Employee Web Panels.
  • Massive reduction in development time and complexity.

Backend API & Logic

Laravel (PHP 8.x)

  • Built-in Auth (Sanctum/Breeze), ORM (Eloquent), and Routing out-of-the-box.
  • Extremely affordable developer ecosystem.
  • Built-in AWS S3 disk driver for instant cloud storage integration.

AWS Infrastructure

EC2 + RDS + S3 (Free Tier)

  • Compute: AWS EC2 (t3.micro) running Laravel (PHP-FPM) and Nginx.
  • Database: AWS RDS (PostgreSQL/MySQL db.t3.micro) for secure, managed data.
  • Storage: Amazon S3 for cheap, infinitely scalable selfie/incident image storage.

3. Feature Breakdown by User Role

Omnichannel Access & Core Capabilities

To ensure no employee is left behind due to device constraints, we've introduced an Employee Web Portal built seamlessly into the Laravel backend, sharing the same logic as the Admin/Client panels.

Employee Mobile App (Android)

  • ✓ Native Start/End Duty with forced GPS checks.
  • ✓ Native Camera integration for verified selfies.
  • ✓ View assigned shifts and site details.
  • ✓ Submit Incident Reports with live photos.
  • ✓ Emergency SOS button for immediate escalation.

Employee Web Portal (Browser)

  • ✓ Browser-based login for users without Androids.
  • ✓ HTML5 Geolocation for duty check-ins (fallback).
  • ✓ HTML5 Media capture for selfies via mobile browsers.
  • ✓ View schedules and basic profile data.
  • ✓ Optimized for low-end mobile web browsers.

Admin Web Panel

  • ✓ Master dashboard with daily attendance overview.
  • ✓ Manage Employees, Clients, and Site locations.
  • ✓ Create and assign shifts to workforce.
  • ✓ Monitor incoming Incident Reports and SOS alerts.
  • ✓ Generate/Download global daily activity reports.

Client Web Panel

  • ✓ Read-only access restricted to own site/team.
  • ✓ Real-time Traffic Light Status (Green/Yellow/Red).
  • ✓ View guard check-in timestamps and selfie proofs.
  • ✓ View incidents relevant to their specific sites.
  • ✓ Download tailored daily PDF reports.

By leveraging Laravel's built-in Blade templating, we eliminate the need for a separate Web SPA. This drastically reduces development hours and QA complexity, further driving down costs.

Full-Stack Development ₹38,000 Dev time drops from 4 weeks to ~3 weeks. Laravel Blade replaces the need to build and integrate a separate Admin web application.
